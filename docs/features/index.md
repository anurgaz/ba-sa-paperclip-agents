# Features — Артефакты агентов

Документация по фичам, сгенерированная BA и SA агентами.

Каждая фича содержит:

- **User Story** (BA Agent) — бизнес-требования, acceptance criteria, edge cases
- **API Spec** (SA Agent) — техническая спецификация endpoint'ов
- **Sequence Diagram** (SA Agent) — диаграммы взаимодействия
- **Test Cases** (SA Agent) — тест-кейсы

---

*Артефакты автоматически публикуются после прохождения валидации (4/4 checks).*
